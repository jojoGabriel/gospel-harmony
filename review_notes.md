# Review Notes

Use this file to capture editorial notes while reading the review edition.

Recommended format for each note:

- `[review ref]` brief note

Optional additions when helpful:

- `Priority:` high / medium / low
- `Action:` revise / verify / expand / trim / reconsider placement

Example:

- `[John 6:22-71]` Bread of Life paragraph still feels compressed near the hard-saying turn.
- `Priority:` medium
- `Action:` expand
